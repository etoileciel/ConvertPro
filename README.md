# ConvertPro
Convertisseur de fichiers universel

Guide complet pour grands débutants
Ce guide vous explique, étape par étape,
comment installer et utiliser ConvertPro.
Aucune connaissance informatique requise !

0. Avant de commencer — ce que vous devez savoir
Pas de panique ! Ce guide a été écrit pour quelqu'un qui n'a jamais installé un logiciel. Chaque action est expliquée clairement. Suivez simplement les étapes dans l'ordre.

Qu'est-ce que ConvertPro ?
ConvertPro est un logiciel gratuit qui vous permet de transformer vos fichiers d'un format à un autre. Par exemple :
⦁	Une photo en format PNG → la transformer en PDF pour l'envoyer par e-mail
⦁	Un document PDF → l'extraire en images pour le modifier
⦁	Une vidéo MP4 → en extraire uniquement le son (MP3)
⦁	Plusieurs PDF → les fusionner en un seul fichier

Ce dont vous avez besoin
✅ Liste de départ
• Un ordinateur sous Windows 10 ou Windows 11
• Une connexion internet active (seulement pour l'installation)
• Le fichier ZIP téléchargé : ConvertPro_Installer.zip
• Environ 10 à 15 minutes de votre temps

ℹ️  Vocabulaire de base
Fichier : un document stocké sur votre ordinateur (photo, vidéo, PDF, musique...)
Dossier : une boîte qui contient des fichiers (comme un classeur physique)
Bureau (Desktop) : l'écran d'accueil de Windows avec vos icônes
Double-cliquer : appuyer deux fois rapidement sur le bouton gauche de la souris
Clic droit : appuyer une fois sur le bouton DROIT de la souris
Télécharger : récupérer un fichier depuis internet pour le mettre sur votre ordi
Installer : préparer un logiciel pour qu'il fonctionne sur votre ordinateur
ZIP : une sorte d'enveloppe compressée qui contient plusieurs fichiers

1. Récupérer le fichier d'installation
Vous avez téléchargé un fichier appelé ConvertPro_Installer.zip. Ce fichier est une "enveloppe" qui contient tous les éléments nécessaires. Avant d'installer quoi que ce soit, vous devez l'ouvrir et en extraire le contenu.

Étape 1 — Trouver votre fichier téléchargé
Quand on télécharge un fichier sur internet, il va en général dans un dossier spécial appelé « Téléchargements ». Voici comment y accéder :

1.	Ouvrez l'Explorateur de fichiers. C'est l'icône en forme de dossier jaune dans votre barre des tâches (en bas de l'écran), ou appuyez simultanément sur les touches Windows + E de votre clavier.
2.	Dans la colonne de gauche, cliquez sur « Téléchargements » (ou « Downloads »).
3.	Vous devriez voir le fichier ConvertPro_Installer.zip dans cette liste.

💡 Astuce — si vous ne trouvez pas le fichier
Dans l'Explorateur de fichiers, cliquez sur la barre de recherche (en haut à droite)
Tapez : ConvertPro_Installer.zip
Appuyez sur la touche Entrée. Windows va le retrouver pour vous.

Étape 2 — Extraire (décompresser) le fichier ZIP
Le fichier ZIP est comme une enveloppe fermée. Vous devez l'ouvrir pour accéder aux fichiers à l'intérieur. Cette opération s'appelle « extraire » ou « décompresser ».

1.	Faites un clic droit sur le fichier ConvertPro_Installer.zip (bouton droit de la souris).
2.	Un menu apparaît. Cliquez sur « Extraire tout... » (ou « Extract All... » si Windows est en anglais).
3.	Une fenêtre s'ouvre et vous demande où extraire les fichiers. Laissez l'emplacement par défaut (il propose souvent le même dossier Téléchargements).
4.	Cliquez sur le bouton « Extraire ».
5.	Après quelques secondes, un nouveau dossier appelé convertisseur apparaît. C'est bon !

⚠️  Attention — ne travaillez pas directement dans le ZIP
Certaines personnes double-cliquent sur le ZIP et voient les fichiers à l'intérieur.
NE lancez pas les fichiers depuis l'intérieur du ZIP — ça ne fonctionnera pas !
Extrayez toujours les fichiers d'abord, puis travaillez dans le dossier extrait.

2. Installer ConvertPro
Maintenant que vous avez le dossier extrait, l'installation est très simple. Un seul double-clic suffit — le programme fait tout le reste automatiquement !

Étape 3 — Ouvrir le dossier extrait
1.	Ouvrez le dossier convertisseur qui vient d'être créé (double-cliquez dessus).
2.	Vous devez voir ces fichiers à l'intérieur :

Nom du fichier	À quoi ça sert ?
INSTALLER_ConvertPro.bat	Le programme d'installation — c'est celui que vous allez lancer
convertisseur.py	Le logiciel lui-même (ne pas supprimer)
LISEZ-MOI.md	Un guide de référence rapide

Étape 4 — Lancer l'installation
C'est ici que la magie opère. Un seul double-clic et le programme fait tout tout seul !

1.	Double-cliquez sur le fichier INSTALLER_ConvertPro.bat (celui avec l'extension .bat).
2.	Windows va peut-être afficher un avertissement de sécurité en bleu ou orange. C'est normal pour les fichiers .bat. Cliquez sur « Informations complémentaires » puis sur « Exécuter quand même ».
3.	Une fenêtre noire avec du texte blanc s'ouvre. C'est la fenêtre d'installation — c'est normal !
4.	Lisez les messages qui défilent. L'installation se passe en 5 étapes clairement indiquées.
5.	Si Python n'est pas sur votre ordinateur, le programme le télécharge et l'installe automatiquement. Cela peut prendre 3 à 5 minutes selon votre connexion internet.
6.	À la fin, la fenêtre affiche « INSTALLATION TERMINÉE ! » et vous demande si vous voulez lancer ConvertPro maintenant.
7.	Tapez la lettre O sur votre clavier (comme dans « Oui ») et appuyez sur Entrée.

✅ Comment savoir si l'installation a réussi ?
Une icône ConvertPro est apparue sur votre Bureau (l'écran d'accueil)
ConvertPro apparaît dans votre menu Démarrer (la touche Windows)
La fenêtre d'installation affiche : INSTALLATION TERMINÉE !

3. Utiliser ConvertPro — le guide complet
Bravo, ConvertPro est installé ! Maintenant voici comment l'utiliser au quotidien.

Ouvrir ConvertPro
Vous avez deux façons de lancer le logiciel :
⦁	Depuis le Bureau : double-cliquez sur l'icône ConvertPro
⦁	Depuis le menu Démarrer : cliquez sur la touche Windows (en bas à gauche), tapez ConvertPro, et appuyez sur Entrée

La fenêtre principale de ConvertPro s'ouvre. Elle est divisée en deux parties :
⦁	À gauche : la liste des types de conversions disponibles
⦁	À droite : les fichiers à convertir et les options de sortie

Comment faire une conversion — étapes universelles
Quelle que soit la conversion que vous souhaitez faire, les étapes sont toujours les mêmes :

1.	Choisissez le type de conversion dans la colonne de gauche en cliquant dessus (exemple : Image → PDF).
2.	Cliquez sur le bouton « + Ajouter » pour sélectionner votre (vos) fichier(s) source.
3.	Une fenêtre s'ouvre pour parcourir vos dossiers. Naviguez jusqu'à votre fichier, cliquez dessus, puis cliquez sur « Ouvrir ».
4.	Si vous voulez choisir où sera sauvegardé le résultat, cliquez sur « Parcourir » à côté de « Dossier de sortie ». Sinon, le fichier converti sera sauvegardé au même endroit que l'original.
5.	Cliquez sur le grand bouton violet « ⟳ CONVERTIR ».
6.	Attendez quelques secondes. Un message « Succès ! » apparaît quand c'est terminé.
7.	Votre fichier converti se trouve maintenant dans le dossier de sortie choisi.

Conversions disponibles en détail

🖼️  Conversion 1 — Image → PDF
Quand utiliser : vous avez une ou plusieurs photos (JPG, PNG...) et vous voulez les mettre dans un seul PDF pour l'imprimer ou l'envoyer par e-mail.
⦁	Formats acceptés en entrée : JPG, JPEG, PNG, BMP, GIF, WEBP, TIFF
⦁	Format de sortie : PDF
⦁	Cas pratique : scanner une facture en photo et la convertir en PDF

📄  Conversion 2 — PDF → Images
Quand utiliser : vous avez un PDF et vous voulez extraire ses pages sous forme d'images.
⦁	Format accepté en entrée : PDF
⦁	Format de sortie : une image PNG par page (ex : rapport_page1.png, rapport_page2.png...)
⦁	Cas pratique : extraire une illustration d'un PDF pour la réutiliser ailleurs
⚠️  Prérequis pour PDF → Images
Cette conversion nécessite un outil supplémentaire appelé Poppler.
Si ça ne fonctionne pas, consultez la section 5 de ce guide (Résolution de problèmes).

🔄  Conversion 3 — Image → Image (changer de format)
Quand utiliser : vous avez une image dans un format et vous voulez la convertir dans un autre format.
⦁	Formats acceptés : JPG, PNG, BMP, GIF, WEBP, TIFF
⦁	Choisissez le format de sortie souhaité dans la zone « Format de sortie » qui apparaît à droite
⦁	Cas pratique : convertir une image WEBP (format web) en JPG pour l'imprimer

↔️  Conversion 4 & 5 — PNG ↔ JPEG
Quand utiliser : raccourcis rapides pour passer de PNG à JPG ou de JPG à PNG.
⦁	PNG → JPEG : réduit la taille du fichier (idéal pour partager sur internet)
⦁	JPEG → PNG : meilleure qualité, garde la transparence

📝  Conversion 6 — Texte → PDF
Quand utiliser : vous avez un fichier texte (.txt) ou un tableau (.csv) et vous voulez le transformer en PDF lisible.
⦁	Formats acceptés : TXT, CSV
⦁	Format de sortie : PDF
⦁	Cas pratique : transformer des notes texte en PDF propre à envoyer

🎬  Conversion 7 & 8 — MP4 → MP3 / AVI (via FFmpeg)
Quand utiliser : vous avez une vidéo et vous voulez en extraire le son (MP3) ou la convertir dans un autre format vidéo (AVI).
⦁	Formats acceptés : MP4, AVI, MKV, MOV
⦁	Cas pratique : extraire la bande son d'une vidéo YouTube téléchargée
⚠️  Prérequis — FFmpeg doit être installé
Les conversions vidéo/audio nécessitent un programme gratuit appelé FFmpeg.
Si cette option affiche une erreur, consultez la section 4 de ce guide.
Toutes les autres conversions fonctionnent sans FFmpeg.

🔗  Conversion 9 — Fusion de PDF
Quand utiliser : vous avez plusieurs fichiers PDF séparés et vous voulez les assembler en un seul.
1.	Sélectionnez « Fusion PDF » dans la colonne de gauche.
2.	Cliquez plusieurs fois sur « + Ajouter » pour ajouter tous vos PDF. Ils apparaissent dans la liste.
3.	L'ordre dans la liste = l'ordre dans le PDF final. Pour retirer un fichier, sélectionnez-le et cliquez « ✕ Retirer ».
4.	Cliquez sur « CONVERTIR ». Le résultat s'appelle fusion_output.pdf.

4. Installer FFmpeg (pour les conversions vidéo/audio)
FFmpeg est un logiciel gratuit et open-source qui gère les fichiers vidéo et audio. ConvertPro en a besoin pour les conversions MP4 → MP3 et MP4 → AVI. Voici comment l'installer :

Télécharger FFmpeg
1.	Ouvrez votre navigateur internet (Chrome, Firefox, Edge...).
2.	Allez sur le site officiel : ffmpeg.org/download.html
3.	Cliquez sur le logo Windows (le logo avec les 4 carrés de couleur).
4.	Cliquez sur le lien « gyan.dev » ou « BtbN » — ces deux sources sont fiables.
5.	Téléchargez la version qui contient le mot « essentials » ou « full » dans son nom, au format .zip.

Installer FFmpeg
1.	Extrayez le fichier ZIP téléchargé (comme vous l'avez fait pour ConvertPro).
2.	Ouvrez le dossier extrait. Vous verrez un sous-dossier avec un nom long (ex : ffmpeg-6.0-essentials_build). Ouvrez-le.
3.	À l'intérieur, il y a un dossier bin. Notez l'adresse complète de ce dossier (exemple : C:\Users\Votre Nom\Téléchargements\ffmpeg-6.0\bin).
4.	Cliquez sur le menu Démarrer (touche Windows), cherchez « Variables d'environnement » et ouvrez l'option « Modifier les variables d'environnement système ».
5.	Cliquez sur « Variables d'environnement... » en bas de la fenêtre.
6.	Dans la section « Variables utilisateur », double-cliquez sur la variable Path.
7.	Cliquez sur « Nouveau » et collez le chemin du dossier bin de FFmpeg.
8.	Cliquez sur OK, OK, OK pour fermer toutes les fenêtres.
9.	Redémarrez votre ordinateur pour que les changements s'appliquent.

✅ Vérifier que FFmpeg est bien installé
Appuyez sur les touches Windows + R simultanément.
Tapez cmd et appuyez sur Entrée. Une fenêtre noire s'ouvre.
Tapez : ffmpeg -version et appuyez sur Entrée.
Si vous voyez une ligne commençant par « ffmpeg version... », c'est bon !
Si vous voyez « La commande n'est pas reconnue », recommencez depuis l'étape 4.

5. Résolution de problèmes fréquents

❌ Problème : La fenêtre noire disparaît aussitôt
Solution : Faites un clic droit sur le fichier INSTALLER_ConvertPro.bat.
Choisissez « Exécuter en tant qu'administrateur ».
Confirmez en cliquant sur « Oui » si Windows vous le demande.

❌ Problème : « Python n'est pas reconnu » après l'installation
Solution : Redémarrez votre ordinateur complètement (pas juste mettre en veille).
Après le redémarrage, relancez l'installateur.
Si le problème persiste, installez Python manuellement depuis python.org/downloads
Cochez ABSOLUMENT la case « Add Python to PATH » pendant l'installation de Python.

❌ Problème : Le raccourci sur le Bureau ne fonctionne pas
Solution : Naviguez manuellement jusqu'au dossier d'installation.
Ouvrez l'Explorateur de fichiers, puis tapez dans la barre d'adresse :
%LOCALAPPDATA%\ConvertPro
Appuyez sur Entrée. Double-cliquez sur ConvertPro.bat pour lancer le logiciel.

❌ Problème : Erreur lors de la conversion PDF → Images
Cette conversion nécessite Poppler, un outil supplémentaire.
Téléchargez Poppler pour Windows sur : github.com/oschwartz10612/poppler-windows
Téléchargez la dernière version Release, extrayez-la, et ajoutez le dossier bin au PATH
(même procédure que pour FFmpeg, décrite à la section 4 de ce guide)

❌ Problème : Erreur « Module not found » ou bibliothèque manquante
Ouvrez l'Explorateur de fichiers, puis dans la barre d'adresse tapez :
%LOCALAPPDATA%\ConvertPro
Appuyez sur Entrée, puis double-cliquez sur ConvertPro.bat.
Si l'erreur persiste, réinstallez en relançant INSTALLER_ConvertPro.bat.

6. Désinstaller ConvertPro
Pour désinstaller ConvertPro proprement de votre ordinateur :

1.	Ouvrez l'Explorateur de fichiers.
2.	Dans la barre d'adresse en haut, tapez exactement : %LOCALAPPDATA%\ConvertPro
3.	Appuyez sur Entrée. Le dossier d'installation s'ouvre.
4.	Double-cliquez sur le fichier « Désinstaller ConvertPro.bat ».
5.	Le programme supprime automatiquement tous les fichiers, le raccourci du Bureau et l'entrée du menu Démarrer.

ℹ️  Les bibliothèques Python restent installées
La désinstallation supprime ConvertPro mais pas Python ni ses bibliothèques.
Si vous souhaitez également supprimer Python, faites-le depuis :
Paramètres → Applications → recherchez « Python » → Désinstaller.

7. Conseils et bonnes pratiques

Avant de convertir
⦁	Faites toujours une copie de sauvegarde de vos fichiers originaux avant de les convertir, par sécurité.
⦁	Vérifiez que le fichier source n'est pas ouvert dans un autre logiciel pendant la conversion.
⦁	Pour les gros fichiers (vidéos HD, PDFs de centaines de pages), la conversion peut prendre plusieurs minutes. Soyez patient !

Pour de meilleurs résultats
⦁	Image → PDF : pour plusieurs images dans un seul PDF, ajoutez-les toutes d'un coup en tenant la touche Ctrl enfoncée pendant que vous les sélectionnez.
⦁	Fusion PDF : l'ordre dans lequel vous ajoutez les fichiers correspond à l'ordre dans le PDF final. Réfléchissez-y avant de convertir.
⦁	Texte → PDF : si votre fichier texte contient des caractères spéciaux (accents, symboles), vérifiez que l'encodage est UTF-8.

Organiser vos fichiers convertis
⦁	Choisissez toujours un dossier de sortie clair et organisé, par exemple un dossier « Convertis » sur votre Bureau.
⦁	Pour la conversion PDF → Images, chaque page crée un fichier séparé. Prévoyez un dossier vide pour ne pas mélanger les fichiers.
